# Orbis Watch Lab

Aplicativo Android para investigação BLE do relógio G28 usando apenas o celular.

Funções iniciais:
- localizar o G28;
- conectar por Bluetooth Low Energy;
- listar serviços e características GATT;
- ativar notificações;
- registrar pacotes recebidos em hexadecimal.

O APK é compilado automaticamente pelo GitHub Actions.
